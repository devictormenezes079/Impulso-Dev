# Impulso-Dev
O Impulso Dev nasceu da ideia de criar um espaço onde desenvolvedores pudessem trocar experiências de um jeito simples e direto. Mais do que um site de posts, é uma rede social técnica onde você monta seu perfil, destaca o que sabe fazer (como aquelas badges de cursos que a gente adora) e interage com a comunidade em um feed dinâmico.
